OUTPUT:

It's just a full width Button.
You create a Block Button by making the width 100%.


Code:

<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="device-width, initial-scale=1">
        <style>
          #blockBtn{
            width:100%;
            padding:10px;
            background:royalblue;
            color:white;
          }
        </style>
    </head>
    <body>
        <button id="blockBtn"> Block Button </button>
    </body>
</html>
