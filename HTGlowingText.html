How To Create Glowing Text



OUTPUT:

Glowing Text

HTML code

<p class="glow"> Glowing Text </p>

CSS code

.glow{
    font-size:xxx-large;
    color:white;
    font-family:cursive;
    padding:20px;
    text-align:center;
    animation:glow 1s ease-in-out infinite alternate;
}
@-webkit-keyframes glow{
    from{
      text-shadow:0 0 10px #fff, 0 0 20px #fff, 0 0 30px yellow, 0 0 40px yellow;
    }
    to{
      text-shadow:0 0 20px #fff, 0 0 30px #fff, 0 0 40px blue, 0 0 50px blue, 0 0 60px blue;
    }
}
