How To Create Dropdown



Create a simple dropdown with CSS
Click the Button below to drop

OUTPUT:

Home About Settings Contact

HTML code

<div id="dropCont">
    <button id="dropBtn"> Drop Down </button>
    <div id="drop">
        <a href="#"> Home </a>
       <a href="#"> About </a>
       <a href="#"> Settings </a>
       <a href="#"> Contact </a>
    </div>
</div>

CSS code

#dropCont{
    text-align:center;
    display:inline-block;
}
#drop{
    display:none;
    position:absolute;
    background:black;
    padding:10px;
    text-align:left;
}
#drop a{
    color:white;
}
#dropBtn{
    background:seagreen;
    color:white;
    padding:5px;
}
#dropCont:hover #drop{
    display:block;
}
