How To Create Popups

Learn how to create a Popup.
Click on the text below.

OUTPUT:

Click me to Popup Popup Text

HTML code

<div id="popClick" onclick ="displayPopup()"> Click me to Popup <span id="popup"> Popup Text </span> </div>

CSS code

#popClick{
    text-align:center;
    display:inline-block;
}
#popup{
    visibility:hidden;
    background:#000;
    color:white;
    position:absolute;
    left:50%;
    top:335px;
    margin-left:-50px;
    padding:5px;
    border-radius:10px;
}

JavaScript code

function displayPopup(){
    let x =document.getElementById("popup");
    if(x.style.visibility=="visible"){
        x.style.visibility="hidden";
    }else{
        x.style.visibility="visible";
    }
}
