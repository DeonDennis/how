How To Create Placeholder Color



You have always wanted to Change the Placeholder color. Here is How To.

OUTPUT:

HTML code

<input type= "text" placeholder= "This is a blue placeholder" id= "holderColor"/>

CSS code

::placeholder{
    color:blue;
    opacity:0.5;
}
