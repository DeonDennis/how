How To Create Tooltip



Learn How to create a too tip with CSS.
Hover over the text below.

OUTPUT:

Hover to show Tooltip Top

HTML code

<div id="TipCont"> Hover to show <span id="Tiptext"> Tooltip Top </span> </div>

CSS code

#TipCont{
    display:inline-block;
    text-align:center;
    border-bottom:dotted 1px blue;
}
#Tiptext{
    visibility:hidden;
    background:#000;
    color:white;
    padding:5px;
    border-radius:5px;
    text-align:center;
    position:absolute;
    top:290px;
    left:50%;
    margin-left:-50px;
    opacity:0.2;
    transition:opacity 1s;
}
#TipCont:hover #Tiptext{
    visibility:visible;
    opacity:1;
}
